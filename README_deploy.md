## Como subir no Vercel (via GitHub)

### 1. Criar repositório no GitHub
- Acesse github.com → New repository
- Nome: `quiz-estilo-50mais`
- Visibilidade: **Public**
- Clique em "Create repository"

### 2. Subir os arquivos
```bash
git remote add origin https://github.com/SEU_USUARIO/quiz-estilo-50mais.git
git branch -M main
git push -u origin main
```

### 3. Conectar ao Vercel
- Acesse vercel.com → Add New Project
- Importe o repositório do GitHub
- Clique em **Deploy**
- Pronto! Em ~1 minuto seu quiz está no ar 🚀

## Estrutura
```
index.html     ← Quiz completo com fotos embutidas
vercel.json    ← Configuração do Vercel
```
